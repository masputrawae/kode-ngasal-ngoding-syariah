# Function Di JavaScript

**Function** adalah blok kode yang dirancang untuk melakukan tugas tertentu, keuntungan menggunakan **function** antara lain:

- Mengorganisir Kode Agar Lebih Rapih
- Bisa digunakan ulang (Mengurangi Redudansi)
- Membuat Program Lebih Modular

Contoh **function** sederhana:

```javascript
function sapa(nama) {
	console.log('Halo, ' + nama + ' Selamat Datang!')
}

// Panggil fungsi
sapa('Joni') // Output: Halo, Joni Selamat Datang!

// Bisa dipanggil lagi
sapa('Jono') // Output: Halo, Jono Selamat Datang!
```
