# Variabel Boolean

[[Variabel]] Boolean adalah sebuah [[Variabel]] yang berisi data Boolean, yaitu hanya berisi antara True (ya) dan False (tidak).

Contoh di JavaScript:

```javascript
const sedangBelajar = true

// Sedikit percabangan
if (sedangBelajar) {
	console.log('Saya sedang belajar')
} else {
	console.log('Saya tidak sedang belajar')
}
```

Kode [[JavaScript]] tersebut akan mengecek [[Variabel]] apakah `true` / `false`, secara `if` tersebut akan mengecek `sedangBelajar` apakah `true` (benar), Jika iya tampilkan di konsol `Saya sedang belajar` jika `false` (tidak) maka tampilkan `Saya tidak sedang belajar`.

Contoh di bahasa Pemrograman [[Python]]:

```python
sedang_belajar = True

if sedang_belajar:
    print('Saya sedang belajar')
else:
    print('Saya tidak sedang belajar')
```

Easy 😉
