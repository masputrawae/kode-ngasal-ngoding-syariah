# Case Sensitive

Contoh langsung begini, `nama` dengan `Nama` itu adalah dua bal yang berbeda, kalau kita bacaan akan sama, cuma kalau di dalam beberapa atau kebanyakan Bahasa Pemrograman dua hal tersebut akan beda, nah itu dia yang disebut Case Sensitive.

contoh kasus di Python

```python
nama = "John"
Nama = "Ipin"

# Coba print masing masing
print(nama) # Output: John
print(Nama) # Output: Ipin
```

kenapa?: kalau seandainya Python tidak Case Sensitive, maka kedua print tersebut sama, berhubung Case Sensitive maka mereka diidentifikasi 2 hal yang berbeda.
